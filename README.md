# ALX Low level programming

This is a repository documenting my time learning C programming at ALX Software Engineering program

## Requirements
- Ubuntu 20.04 LTS
- gcc 4.8.4 (-Wall, -Werror, -Wextra, -pedantic -std=gnu89)
- Allowed editors: `vi, vim, emacs`
- Code should use `Betty` style.
